# Change Log

## 0.1.0 - April, 13th 2026

- Added support for `suite` and `test` as aliases for `describe` and `it`
- Added support for `specify` as an additional `it`-style keyword (Mocha)
- Added support for `xdescribe`, `xit`, and `xtest` — recognized automatically as skipped blocks
- Playwright's `test.describe(...)` is now correctly shown as a group in the outline

## 0.0.3 - April, 10th 2026

- [#1](https://github.com/Ranitzky/test-outline/issues/1) fixed issue, where labels were not shown, if they are not on same line as e.g. `it`
